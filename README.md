# CNN Emotion Detector

## Features

- Facial Emotion Recognition
- CNN Deep Learning Model
- Real-Time Webcam Detection
- Streamlit Web Application
- FER2013 Dataset

## Emotions

1. Angry
2. Disgust
3. Fear
4. Happy
5. Neutral
6. Sad
7. Surprise

## Install

pip install -r requirements.txt

## Train Model

python train.py

## Webcam

python app.py

## Streamlit

streamlit run streamlit_app.py
